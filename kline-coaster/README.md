# K 線雲霄飛車 · Candlestick Coaster

把股票 K 線圖當成雲霄飛車來騎的**第一人稱（POV）3D 網頁遊戲**。
台車坐在收盤價曲線上，重力驅動 —— 下殺俯衝、上漲爬坡，挑戰最大「尖叫值（G 力）」。
純前端、無建置步驟、Three.js 內嵌，可直接丟上任何靜態主機。

## 玩法
- **油門**：按住 空白鍵 / ↑ / 右下「油門」鈕。**有能量限制**（藍色條），用完要放開回充
- **煞車**：按住 ↓ / B / 左下「煞車」鈕。**有限、會過熱**，過熱會鎖死到冷卻過半
- **發薪日**：每撐過 30 天，油門能量補滿、煞車立即冷卻（💰 發薪日！）
- 真重力物理：**上坡掉速、下坡加速、動量延續**；軌道高低 = 收盤價起伏
- **馬達很弱**：陡坡沒催油門會**拋錨停擺 → GAME OVER**，所以油門是生存必需（不是放著就能跑完）
- **隨機背景**：每段行情隨機抽 夜晚 / 日出 / 白天 / 黃昏 / 月夜，含太陽月亮、雲、星空（同 seed 同天色）
- 加速度劇變（俯衝、急彎、被甩）會有**尖叫聲**
- 軌道**左右轉彎**（行情動能 + 隨機），過彎鏡頭內傾
- **出軌 = GAME OVER**：過彎太快（橫向）或衝太快過山頂（被甩飛）→ 翻車動畫
- 目標：活越多天越好；**撐過天數 = 分數**。每支標的用它的**完整歷史長度**（數千個交易日），跑完整條 = 神級，大多數人會中途翻車/拋錨

## 輸入代號即時生成賽道
開始畫面有搜尋框：輸入代號 → 即時抓日K → 生賽道。
- **加密貨幣**：直接打 Binance 公開 API（瀏覽器可直連、免金鑰、免後端），輸入 BTC/ETH/SOL… 立刻可玩。注意 Binance 在部分地區（如美國）會地理封鎖，未來可加 CoinGecko 備援。
- **股票**：走自家代理 `functions/api/quote.js`（伺服器端抓 Stooq，避開 CORS）→ **需部署後端才生效**；未部署時會 fallback「用代號生成示意賽道」。
- **每月賽季**：抓的資料一律截止到**上個月底**（整個月固定不變），排行榜 key 帶月份（如 `BTC|2026-06`）→ **每月自動洗榜重來**。好處：不必即時抓最新、同月內人人同一份資料（公平），又有每月新賽季的回訪動機。輸入查詢的幣因此也能正式排行。
- 若之後加 CSP，記得允許 `connect-src` 包含 `api.binance.com` 與自身網域。

## 標的關卡 + 排行榜
- 開始畫面可選股票/加密貨幣當關卡（`js/catalog.js`）。**同一支標的 = 固定 seed → 軌道（轉彎/起伏/天色）對所有玩家完全相同**，所以排行才公平。
- 排行榜採**混合制**：跑到終點的人比「**完賽時間**」（計時賽，越快越強）、排在最前；沒跑到的人比「**撐過天數**」（越遠越前）排在後面。這樣高手分得出高下、一般玩家也都被排名。玩家留名字。先用本地（localStorage），部署後端後自動升級成全球榜（見 `humanpending.md` #7、`functions/api/scores.js`）。
- 換真實日K：在 catalog 該標的加 `dataUrl` 指向 `data/symbols/XXX.json`（見 humanpending #8）。
- ⚙️ 調整面板會影響賽道公平性 → 排行模式進場會自動 `resetConfig()` 還原預設。

## 軌道旁廣告看板
軌道左側有白色大型看板（`CONFIG.billboard` 可調數量/大小）。之後業配時呼叫 `game.setBillboardImage(圖片網址)` 即可整批貼上。終點有一面「終點 FINISH」牌（內容待定），`game.setFinishImage(網址)` 可換圖。

## 好改的設定
所有手感參數集中在 `js/game.js` 最上面的 `CONFIG` 物件（台車、賽道、物理、油門/煞車、獎勵、出軌門檻、天際線、音效），改數字即可調整，不用動邏輯。

遊戲中右上角 **⚙️ 調整** 會開啟**即時調整面板**：物理 / 油門煞車 / 相機 / 出軌難度 / 音效 拉 slider **馬上生效**；賽道與天際線類（起伏、轉彎、貼合度、天數…）改完按「套用並重建賽道」即可看效果。

## 部署快取注意
本機 `python3 -m http.server` 不送 Cache-Control，瀏覽器會用啟發式快取，更新 JS 後可能要強制重新整理（Cmd/Ctrl+Shift+R）。正式部署到 Cloudflare Pages / Netlify 時，建議對 `*.js`/`*.css` 設**檔名雜湊或版本查詢字串**（或短 max-age + 重新驗證），避免使用者更新後拿到舊檔。
- 可調**波動度**生成不同刺激度的隨機行情（也會改變坡高）
- 可**載入自己的 K 線**（CSV / JSON）把任何股票變成一條軌道
- 結束後可**分享成績**，連結帶 `?seed=` 讓朋友挑戰同一段行情

## 手機
支援手機直/橫向：版面自適應，控制改用左下煞車 / 右下衝刺觸控鈕，已停用雙擊縮放與下拉刷新。用手機瀏覽器開部署後的網址即可玩。

## 本機執行
沒有任何建置步驟，但因為用了 ES module + fetch，需要透過 HTTP 伺服器開啟（不能直接雙擊 file://）：

```bash
cd kline-coaster
python3 -m http.server 8000
# 瀏覽器開 http://localhost:8000
```

## 部署（靜態主機）
整個資料夾就是成品，直接上傳即可。推薦免費方案：
- **Cloudflare Pages** / **Netlify** / **Vercel** / **GitHub Pages**
- 把資料夾拖上去或連 Git repo，build command 留空，輸出目錄設為這個資料夾。

## 接真實股價資料
目前資料有三條路：
1. **內建生成器**（隨機行情，可調波動度、可用 seed 重現）
2. **使用者上傳** CSV / JSON（前端解析，零後端）
3. `data/sample.json` 範例（DEMO）

要接「即時 / 指定股票」需要資料來源 API（多半要金鑰、且有 CORS / 授權限制）—— 見 `humanpending.md`。
CSV 欄位只要包含 open / high / low / close（中英不分大小寫皆可）即可；
JSON 可為 `[{o,h,l,c}, ...]` 或純收盤價陣列 `[100, 102, 98, ...]`。

## 廣告變現
版位已預留，找 `index.html` 裡的兩個註解區塊：
- `#ad-top`：頂部 728×90 橫幅
- `#ad-end`：結束畫面 300×100

把 placeholder 換成你的 AdSense（或其他聯播網）代碼，並在 `<head>` 加上聯播網的載入 script。
注意：AdSense 需要**已上線的網域 + 內容審核通過**才會發放廣告 —— 見 `humanpending.md`。

